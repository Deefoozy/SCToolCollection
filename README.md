# sc_helper_funcs

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```

### Run Unit Tests

```sh
# Run tests in watch mode (development)
npm run test

# Run tests once
npm run test:unit

# Run tests with coverage report
npm run test:coverage
```
